# Front-End Dev Quiz for Viget Labs #

## Install Middleman ##

    $ gem install middleman

## Clone Repo ##

    $ git clone git@github.com:courington/viget_fed_quiz.git
    $ bundle install

## Run Middleman ##

    $ middleman
